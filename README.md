# League of Legends Champions

Showcase Project - React, TypeScript and Material-UI with styled-components

## Demo

Project is available at [Link](https://league-of-legends-psi.vercel.app/)

## Start

Do for both directories:

```
yarn
yarn start
```

or

```
npm i
npm run start
```
